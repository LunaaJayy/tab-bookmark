### Iteraties
Dit is alle informatie die ik heb opgeschreven:

Berend gebruikt schermlezer, ook kleuren om het beter te zien en de zoomfunctie. Zijn muis is heel groot. De kleur is een soort border om de tekst die wordt gelezen, als een soort border. Hij gebruikt NVDA screen reader.

***Wil je dat je meteen naar het geselecteerde element gaat, of pas bij het indrukken van een toets?***

Instelbaar het liefst → ‘wil je naar je opgeslagen bookmark?’ → berend wil dat allebei

***Wil je daarna naar het volgende element gaat in de focus lijst, of liever beginnen bij het begin daarna? Denk gewoon verder maar you never know.*** 

Verder gaan. 

***Welke letter is handig om te gebruiken als shortcut? Q? omdat dat naast de tab zit, weet je wat er wordt bedoelt als je de letter Q zegt? Gebruiken slechtzienden een ander soort toetsenbord?***

ctrl+shift+y

**Verder:**
- Schermlezers vangen dingen zoals alleen nummers niet op, schrijf het dan volledig uit.
- De H1, dus hetgeen wat het eerst wordt gelezen laat blijken wat het is.
- De extra content, dus wat niet een hoofd onderwerp ding is, niet een apart *tab* dingetje maken.
- Stop en start buttons kan beste zelfde button zijn (dus toggle)
- Feedback!!!! Bij dingen die gebeuren.
- Dingen naast elkaar is beter voor slechtzienden, ze zoomen het scherm in dus dingen naast elkaar is beter.
- **Websites hebben vaak dat als ze laden, wat er in de header staat, meteen wordt opgelezen??**
- Ranges in geluid mogen best laag → je probeert jezelf als slechtziende al niet te overprikkelen
- Input dingen en teksten erg beschrijvend typen
- Denk aan contrast!
- ‘&’ en ‘home’ gebruiken slechtzienden om naar begin en eind van de pagina te gaan.
- Kijk naar NVDA lijst sneltoetsen die al bestaan → misschien addon maken?

**De iteraties**
De iteraties die ik aan mijn ontwerp wil doen is dat ik sowieso meer feedback wil toevoegen aan elke stap die je zet bij mijn ontwerp. Ik heb dus bijv een command die je in moet toetsen om een focus bookmark op te slaan voor de screenreader, als deze wordt ingedrukt zal er een stem klinken die zegt: 'opgeslagen!'. Daarnaast ga ik een toggle knop maken waarbij je kan instellen of je bij een webpagina meteen naar de bookmark gaat, of dat je bij het begin van de pagina wil beginnen. Dus dat er nog een extra shortcut komt waarbij je meteen naar de focus toe kan gaan. Ook wil ik extra contrast toevoegen bij de popup.
